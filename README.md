# AI-Powered Strategy Report

## Features
- Generates 2-3 page market insights report using GPT-4
- Highlights competitors, customer trends, and actionable strategies
- Saves report to a text file

## Setup
1. Create virtual environment using below commands
   `python -m venv venv`
   `venv\Scripts\activate`
2. Install dependencies:
pip install -r requirements.txt
3. Add your OpenAI API key in `config.py`.
4. Run:
python generate_report.py
